# Select-next-quotes
Select next quotes - plugin for Sublime Text 3

## Usage
* <kbd>Ctrl+Alt+Right</kbd> to select next double-quoted content
* <kbd>Ctrl+Alt+Left</kbd> to select previous double-quoted content
